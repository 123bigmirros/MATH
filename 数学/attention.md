# 极限
- 要注意题目给的是$\infty \ -\infty\  +\infty$这三者在求极限时是有区别的
- $n \rightarrow \infty 时，暗指 n \rightarrow +\infty$
- $$f(x) 可导 \nRightarrow f'(x)连续$$
**当题目告诉你n阶可导时求极限时，只能用到n-1阶，因为不能确定函数在n阶导数之后还是连续的，但是当他说n阶连续时，可以用到n阶**
